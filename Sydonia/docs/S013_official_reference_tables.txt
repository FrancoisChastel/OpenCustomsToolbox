I- INTRODUCTION

        This document encompasses all the Reference Tables belonging to
ASYCUDAWorld. For each table, the fields, field lengths, keys and descriptions are
specified.

II- THE REFERENCE TABLES SUMMARY

       Following is the list of the reference tables in alphabetical order. The column
“Table Name” takes the reader to the respective table section.
Note : All the Reference Tables names are starting with "UN" (Except LogTable), replace
"xx" by "UN" for the complete name.
       All reference tables (Except LogTable) have two fields VALID_FROM and
VALID_TO that define its validity in time. This icon indicates if a field is a primary
key or not.
